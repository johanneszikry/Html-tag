<audio> – Audio
<iframe> – Inline frame
Lists
<ul> – Unordered list
<ol> – Ordered list
<li> – List item
<dl>, <dt>, <dd> – Definition list

<table> – Table
<tr> – Table row
<th> – Table header cell
<td> – Table data cell
<caption> – Table caption
Forms & Inputs
<form> – Form container
<input type="text"> – Text input
<textarea> – Multi-line text
<button> – Button
<select>, <option> – Dropdown list
Semantic Elements
<header> – Header of page/section
<main> – Main content
<section> – Section block
<article> – Article content
<footer> – Footer content
